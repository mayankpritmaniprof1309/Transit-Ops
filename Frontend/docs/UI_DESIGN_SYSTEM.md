=========================================================
GLOBAL UI DESIGN SYSTEM
=========================================================

For every frontend component, page, modal, form, table, chart, navbar, sidebar, and dashboard, follow ONE consistent premium design language.

The application should resemble a modern Enterprise SaaS platform similar to:

• Linear
• Stripe Dashboard
• Vercel
• Notion
• SAP Analytics Cloud
• Oracle Fleet Management
• Uber Freight Dashboard

The UI should feel clean, premium, spacious, and suitable for a real logistics company.

Avoid colorful, cartoonish, or overly decorative interfaces.

=========================================================
BACKGROUND
=========================================================

Use a soft gradient background with subtle blurred blue and purple gradient blobs.

The background should feel elegant and modern without distracting from the content.

Never use busy textures or strong patterns.

=========================================================
GLASSMORPHISM
=========================================================

Use glassmorphism only where appropriate.

Apply Glass Effect To:

✓ Navbar

✓ KPI Cards

✓ Search Bar

✓ Filter Panel

✓ Login Card

✓ Profile Dropdown

✓ Notifications Panel

Do NOT apply glass effect to:

✗ Tables

✗ Forms

✗ Charts

✗ Sidebar

✗ CRUD Pages

=========================================================

Every newly generated React component must follow this design system automatically.

Every page should feel like part of the same premium Enterprise SaaS application.

Whenever generating any frontend code, strictly follow this design system without exception.
